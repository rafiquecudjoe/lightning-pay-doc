Invoices

Invoice operations

Create a new Lightning Network invoice

Creates a new lightning network invoice that can be paid with Bitcoin. The invoice will be valid for 7 days by default, or until the specified expiration time.

Authorizations:
x-api-keybearer
header Parameters
X-API-Key
required
string
Example: pk_test_123abc456def

API key for application identification

Request Body schema: application/json
required
satoshiAmount
required
number <int64> >= 1000

Amount to be paid in satoshis

description
required
string <= 500 characters

Description of the invoice that will be shown to the payer

reference
string <= 50 characters

Custom reference ID for your system to track this invoice

expiresAt
string <date-time>

Expiration time for this invoice in ISO8601 format. Defaults to 7 days if not provided

Responses

Production

https://ejara.io/redocusaurus/api/v1/invoices

Sandbox

https://ejara.io/redocusaurus/api/v1/invoices

Request samples

  • Payload
Content type
application/json
{
  • "satoshiAmount": 50000,
  • "description": "Payment for order #12345",
  • "reference": "ORD-12345-ABC",
  • "expiresAt": "2025-04-20T23:59:59Z"
}

Response samples

  • 201
  • 400
  • 401
  • 500
Content type
application/json
{
  • "message": "Successful",
  • "data": {
    • "invoiceHash": "273b1f8e082811f69f2fe2adbe8d42561e9cc3df6c063c9c21e8033ba586019a",
    • "description": "Payment for order #12345",
    • "status": "pending",
    • "amount": 50000,
    • "expiryDate": "2025-04-22T14:30:00Z",
